# 📄 MultiAgentPPT 之 自动生成PPT大纲

本项目基于 FastMCP 与 A2A 多智能体框架，自动生成结构化的演示文稿大纲（如调研报告、技术汇总等），支持流式SSE输出。

---

## 🧪 一、MCP 工具测试

1. 检查配置文件：

   ```bash
   cat mcp_config.json
   ```

2. 启动 SSE 模式的 MCP 服务：

   ```bash
   cd backend/slide_outline
   fastmcp run --transport sse mcpserver/rag_tool.py
   ```

3. 启动主服务程序：

   ```bash
   python main_api.py
   ```

---

## 📤 二、A2A Client 请求后的输出（`a2a_client.py`）
